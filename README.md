# ⚙️ Dotfiles-StringRam ⚙️
Custom dotfiles for my hyprland setups, includes userChrome file for firefox customization, installation scripts for each one plus other relevant files.

## Currently available themes:
 - Dwarf-FortressLB
 - Catppuccin-Library
 - Stylized-Ruins
 - Basic Firefox CSS
